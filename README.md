# Personal Finance Dashboard

A data visualization mini project built using Python, Pandas, and Matplotlib to analyze spending habits.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Visualizations
- Category-wise spending analysis
- Expense distribution pie chart
- Monthly spending trend
- Savings vs Shopping comparison
- Category trends over months

## Key Insights
- Savings had the highest allocation
- Shopping expenses increased over time
- Entertainment spending was lowest
